# ms.auth

Microservicio de seguridad de la plantilla `SEGURIDAD-EXA`.

Este servicio maneja autenticacion, roles, permisos y emision de JWT. Esta pensado para ejecutarse con Docker Compose desde la raiz del proyecto.
